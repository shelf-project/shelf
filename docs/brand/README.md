# Shelf — Brand Assets

Locked primary mark and favicon for the Shelf project. Use these files; do not regenerate.

## Files

- `shelf-logo.png` — **Primary mark.** Tier-ordered icon + wordmark. Use in READMEs, docs headers, presentation title slides, and anywhere the project needs to be identified by name.
- `shelf-favicon.png` — **Icon-only.** Use as the favicon, app icon, social-card thumbnail, and any context where the wordmark would be illegibly small (≤ 64 px).

## Palette

The mark uses three fixed colours; do not substitute:

- `#1B2333` — slate-navy (background / wordmark on light)
- `#2EC4B6` — teal (primary tier accent)
- `#F4A261` — warm coral (secondary tier accent)

## Usage rules

Do not modify or recolour the mark without maintainer approval.
